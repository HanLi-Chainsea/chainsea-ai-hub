# Case Library

Case library 用來保存「這套 AI workflow 實際怎麼幫到專案」。它面向工程師，也面向主管或跨部門同事，所以內容要比單純技術筆記更重視問題、解法、結果與可複用資產。

## Case Format

使用 [templates/case-study-template.md](../../templates/case-study-template.md)。

每個 case 至少包含：

- Problem
- Solution
- Tools Used
- Result
- Reusable Assets
- Date / Timeframe
- Owner
- Follow-up
