# ChainSea AI Hub Taxonomy

Use one main category and optional secondary categories. Add tags only when they improve discovery.

## Main Categories

### Agent Skills

Reusable AI-agent workflows, skill packages, Claude Code skills, prompt workflows, or task-specific instructions.

### AI Coding Tools

Tools for code generation, refactoring, code review, local coding assistants, IDE-integrated AI, and software engineering automation.

### Testing & QA

Unit testing, integration testing, E2E testing, test generation, coverage, mutation testing, visual testing, and QA case generation.

### MCP / Tool Integration

MCP servers, API wrappers, tool-calling infrastructure, Jira / GitLab / GitHub integrations, browser automation, database tools, and internal system adapters.

### Frontend Engineering

React, TypeScript, UI testing, component libraries, Figma-to-code, design systems, frontend automation, and browser-based workflows.

### Backend / DevOps

API development, database tooling, Docker, CI/CD, logging, monitoring, deployment automation, and service reliability.

### Research / Crawling / Knowledge Gathering

Web crawling, summarization, research agents, RAG, document parsing, search workflows, and knowledge extraction.

### Documentation / PRD / PM

PRD generation, meeting notes, technical documentation, Jira ticket generation, planning workflows, and product analysis.

### LLM Evaluation

Eval frameworks, benchmark tools, prompt evaluation, regression testing, model comparison, and quality measurement.

### Case Studies

Internal examples of how a tool, skill, or workflow was used in a real project.

### Development Experience

Time-aware engineering notes, implementation retrospectives, trade-off records, and iterative learnings.
