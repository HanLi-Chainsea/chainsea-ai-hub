# Repo Entry Template（類別頁卡片）

> 這是類別頁（如 `resources/repos/ai-agents.md`）使用的精簡卡片格式。
> Detail page 請使用 [templates/resource-card.md](resource-card.md)。

```md
### {{repo_name}}

- Status: Experimental / Useful / Production-used / Deprecated
- Category:
- Best for:
- Why listed:
- Not for:
- Last checked:
- Details: [path/to/detail-page.md](path/to/detail-page.md)
- Link:
```
